Документация по API

Чтение данных

Наш сайт предоставляет открытый доступ для получения всех данных по товарам, для этого воспользуйтесь адресом:

  • api/v1/book_list/

Фильтрация данных доступна по категориям, для этого к адресу добавьте параметр "category" и его slugify:

  • api/v1/book_list/category/iskusstvo

Для того, чтобы взять отдельную запись, необходимо добавить параметр ID товара, который вы хотите получить:

  • api/v1/book_list/id/2725692

Авторизация

Для аутификации отправьте POST запарос с такими параметрами Emal и Password, воспользовавшись адресом /auth/token/login/
После успешной авторизации вам будет предоставлен токен, который в дальнейшем нужно будет указывать в заголовке
Для этого укажите ключ: Authorization и значение: Token *ваш токен*

Управление данными

Для удаления или редактирования данных необходимо обладать провами администартора или быть их создателем
Если вы обладаете такими правами, пройдите авторизацию, после чего вам будут доступные следующие адреса:

  • api/v1/book_destroy/*id* - удаление записи

  • api/v1/book_update/*id* - обновление записи

  • api/v1/book_create/ - создание записи

Как вы могли заметить, обращаться к записи можно только по ее id
